# TopRecomendações

TopRecomendações is a personal web app built with Laravel, Inertia, and Vue.js. It allows users to curate and manage their own 'Top Lists' of favorite restaurants and hotels, offering a clean dashboard to add, rate, and track personal reviews.

## Key Features

* **Personalized Rankings**: Organize your favorite restaurants and hotels with custom ratings.
* **Interactive Dashboard**: A sleek, dark-mode dashboard that displays your top picks at a glance.
* **Full CRUD Functionality**: Easily create, read, update, and delete your evaluations.
* **Responsive Design**: Mobile-first interface built with Tailwind CSS.

## Tech Stack

* **Backend**: Laravel
* **Frontend**: Vue.js 3, Inertia.js
* **Styling**: Tailwind CSS
* **Database**: MySQL
